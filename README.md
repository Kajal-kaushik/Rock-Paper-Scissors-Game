# Rock Paper Scissors ✊📄✂️

A simple Rock Paper Scissors game built using **HTML, CSS, and JavaScript**. The player competes against the computer, and the winner is decided based on the classic game rules.

## 🚀 Features
- Player vs Computer
- Random computer choices
- Live score updates
- Winner announcement
- Responsive and interactive interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure

```text
rock-paper-scissors/
│── index.html
│── style.css
│── script.js
```

## ▶️ How to Run
1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Enjoy the game!
